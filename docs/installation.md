# Installation

## Using pip

The recommended way to install LlamaKv is via pip:

```bash
pip install llamakv
```

## From Source

You can also install LlamaKv from source:

```bash
git clone https://github.com/llamasearchai/llamakv.git
cd llamakv
pip install -e .
```

## Dependencies

LlamaKv requires:

- Python 3.8+
- Other dependencies are automatically installed by pip
